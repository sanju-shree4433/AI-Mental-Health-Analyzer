# AI-Mental-Health-Analyzer
AI-Driven Mental Health Risk Prediction from Social  Media Text: A Hybrid ML–DL Approach 

## Overview
This project predicts mental health risk from text using Machine Learning and Natural Language Processing.

## Features
- Mental health risk prediction
- Emotion detection
- Confidence score
- Keyword extraction
- Streamlit web application

## Technologies
- Python
- Streamlit
- Scikit-learn
- NLTK
- TF-IDF
- Logistic Regression

## Dataset
Suicide Detection Dataset from Kaggle

## Run the Project

```bash
pip install -r requirements.txt
streamlit run app.py
```
